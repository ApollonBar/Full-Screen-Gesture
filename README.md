# Full-Screen-Gesture

使用手势助手可以通过手势非常方便地实现返回上一级、返回桌面、切换最近任务等，左右手均可方便地操作手机，再也不怕按不到大屏手机的返回键了

[试用下载](https://github.com/wutongke/Full-Screen-Gesture/releases/download/v1.1/app-release.apk)
